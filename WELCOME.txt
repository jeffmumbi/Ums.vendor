### How to: Add a Vendor Issue
1. Click "Vendor Issues" in sidebar
2. Click "+ Add Issue" button
3. Fill in all fields:
   - Vendor name
   - System affected
   - Description
   - Cause
   - Impact level
   - Downtime hours
   - Root Cause Analysis
   - Status
   - Resolution notes
4. Click "Save Issue"
5. Issue appears in table
6. You can edit or delete it anytime

### How to: Set Password Reminder
1. Click "My Passwords" in sidebar
2. Click "+ Add Password Reminder"
3. Enter:
   - System/Service name (e.g., Email)
   - Username/Account
   - Expiration date (click calendar)
   - Expiration time
   - Notes (optional)
4. Click "Schedule Password Reminder"
5. You'll get alerts starting 7 days before
6. **2-day reminder is the main one** to update it

### How to: Check Notifications
